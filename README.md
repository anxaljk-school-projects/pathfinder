# Pathfinder (Maze Search Problem)

## Run the project

1. Open your terminal
2. Make sure you have [Node.js](https://nodejs.org/en) installed
3. Change into the directory of this project (`.../pathfinder`)
4. Enter the following commands in order:
    ```bash
    npm install
   
    npm run dev
    ```
5. Open the displayed url (probably http://localhost:5173/) in your browser

> Hint: If you want to change the maze's size or other settings, open the file `src/app/environment.ts` and change the values of the variables you wish to adapt.
